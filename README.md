# Buddy
This repository includes source code and firmware releases for the Prusa CORE One Mini (Community Edition) based on a 32-bit ARM microcontroller.

The currently supported model is the Prusa CORE One Mini only

## Resources
- [Prusa CORE One Mini (Community Edition) project files on Printables](https://www.printables.com/model/1734194-prusa-core-one-mini-community-edition-diy)
- [Original Prusa Firmware Buddy](https://github.com/prusa3d/Prusa-Firmware-Buddy/)

## Support
If you want to learn more about the project, make sure to checkout the main post "[CORE One Mini - The Rocky Road to a Prototype.](https://www.printables.com/article/core-one-mini-the-rocky-road-to-a-prototype-Bd3z3a5)" for the complete background and FAQ.

If you intend to buy a Prusa machine, you can use the [referral code](https://help.prusa3d.com/article/p2p-prusa-to-prusa-community-reward-program_670212) "@suit" at checkout in the Prusa online shop. This will give you some Prusameter points and 1 kg of free filament and i'll get some points aswell (which I can use for some free filament). Or you can just download, like and make some of my models here on Printables and even consider to become a [member in my Printables club](https://www.printables.com/@suit#join.@suit.574).

I've also set up a [GoFundMe campaign](https://gofund.me/85fb63e14) to tackle the costs. Thanks to everybody who has donated to the project already - that means a lot and helps me cover the expenses.
